# Other APIs

Use these guides for supporting cuVS APIs that are useful across clustering, indexing, and evaluation workflows.

- [Dynamic Batching](dynamic_batching.md): collect many concurrent small ANN searches into larger GPU search batches.
- [K-selection](select_k.md): select the top `k` values or nearest candidates from larger result sets.
- [Pairwise Distances](pairwise_distances.md): compute distances between vectors for analysis, validation, or algorithm building blocks.
